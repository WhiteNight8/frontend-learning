## Flex布局

水平垂直布局方案

### flex container

特点

1. flex item的布局受flex container属性的设置进行控制和布局
2. flex item不再严格区分块级元素和行内元素
3. flex item默认包裹内容， 但是可以设置宽度和高度

 值 display

1. flex
2. inline-flex

2. flex-wrap

4. align-items
