## 冒泡排序的实现

- 比较两相邻的元素，一轮后，找到最大值，下轮就不用考虑这个最大值 

```typescript
function bubbleSort(arr: number[]): number[] {
    const n = arr.length
    
   for( let i = 0; i < n; i++) {
        for( let j = 0; j < n -i; j++) {
      		if (arr[j] > arr[j+1])  {
           	 swap(arr,j,j+1)
      }
    }
   }
    return arr
}

function swap(arr :number[], i : number, j : number) {
    const temp = arr[i]
    arr[i] = arr[j]
    arr[j] = temp
}
```
