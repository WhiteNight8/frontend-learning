## Map和Set映射和集合

### Map

- new Map（）
- map.set(key,value)
- map.get(key)
- map.has(key)
- map.delete(key)
- map.clear()
- map.size



允许任何类型的键

键不会被转换成字符串，可以是任何类型

### Map迭代

- keys
- values
- entries

都返回一个可迭代对象

### Object.entries：从对象创建Map

### Object.fromEntries: 从Map创建对象

## WeakMap和WeakSet弱映射和弱集合

与map，对象不同，WeakMap不会阻止垃圾回收机制对作为键的对象的回收

### WeakMap

键必须是对象，不能是原始值

不支持迭代

不能准确的直到何时会被回收。由JavaScript引擎决定的

主要应用场景，就是额外的数据，缓存

### WeakSet

也不能迭代，只能添加对象
